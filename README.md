# Optimisation du site web - P4 OpenClassrooms

## Auteur
Ricardo Sandjon

## Description
Projet d'optimisation d'un site web réalisé dans le cadre du Projet 4 OpenClassrooms.  
Le projet comprend :  
- Code optimisé pour desktop  
- Audit Lighthouse et accessibilité  
- Rapport d'optimisation avec captures d'écran  

## Installation
Pour voir le projet, cloner le repo et ouvrir `index.html` dans votre navigateur.
